## Arithmetic operations

- They are the same arithmetic operations we have in math:

    ```python
    # addiction
    print(30 + 4) # (1)!

    # subtraction
    print(30 + 4) # (2)!

    # multiplication
    print(30 * 4) # (3)!

    # float division
    print(30 / 4) # (4)!

    # integer division
    print(30 // 4) # (5)!

    # modulis
    print(30 % 4) # (6)!

    # power
    print(30 ** 4) # (7)!
    ```
